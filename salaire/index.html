Salaire


Dans le contexte d'une entreprise, vous allez créer un script permettant d'afficher les informations de l'utilisateurs et de calculer son salaire. Le tableau des salariés est fourni.

Le salaire brut du salarié figure dans le tableau mais votre affichage doit inclure le salaire net en respectant les règles suivantes:

Taxes :
Le Salarié se voit retrancher à son salaire :

  • 15% d'impôts
  • 5% pour la retraite
  • 7% pour l'assurance maladie

Primes :
Le salarie peut recevoir une prime de 100€ ou une allocation expcetionnelle de 150€.

Déductions : En fonction de divers éléments, le travailleur paie moins d'impôts :

  • 2% de déduction si l'âge est inférieur à 30 ans
  • 1% par personne à charge jusqu'à 6% maximum
  • 5% si il est travailleur handicapé
