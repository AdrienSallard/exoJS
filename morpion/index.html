Morpion


Le jeu du morpion se déroule dans un tableau de 3 x 3, chaque utilisateur possède un signe (croix et rond). Le but est d'aligner 3 signes identiques.

Réaliser le script permettant de joueur à ce jeu contre une autre personne. A chaque clique sur une case, un signe apparait et un message apparait lors de la victoire ou d'une égalité.
